# SPOON Layout

Лендинг-страница для бизнес-шаблона SPOON (Propel).

## Описание

Современный одностраничный сайт с адаптивным дизайном, включающий:
- Шапку с навигацией и логотипом
- Главный баннер с призывом к действию
- Секцию с карточками услуг
- Премиум-секцию с изображениями
- Футер с социальными ссылками

## Структура проекта

```
SPOON/
├── index.html          # Главная страница
├── styles/
│   ├── style.css       # Основные стили
│   └── normalize.css   # Сброс стилей
├── images/             # Изображения секций
├── icons/              # Иконки и иконки карточек
└── logos/              # Логотип компании
```

## Технологии

- HTML5
- CSS3 (с анимациями)
